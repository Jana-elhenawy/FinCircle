# 📌 FinCircle

> *Your financial world in one circle.*

A modern **FinTech web application** designed to help users manage their finances, invest smarter, and connect with others through a collaborative financial ecosystem.

---

## 🚀 Overview

FinCircle is an all-in-one financial platform that combines **personal finance management, AI-powered insights, and peer-to-peer services** into a single intuitive experience.

The goal is to simplify financial decision-making while introducing innovative features like **community lending and financial health tracking**.

---

## ✨ Features

### 💰 Financial Dashboard
- View balance, spending, and financial overview
- Track transactions easily

### 🤖 AI Financial Advisor
- Smart recommendations based on user behavior  
- Helps users make better financial decisions  

### 📈 Investment Options
- Explore investment opportunities inside the app  
- Beginner-friendly investment flow  

### 🤝 Peer-to-Peer Lending
- Users can lend and borrow money from each other  
- Builds a financial community  

### ❤️ Financial Health Score
- Overall evaluation of user's financial status  
- Based on spending, saving, and activity
